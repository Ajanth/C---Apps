any number of folders can be locked safely at the same time.
However this application is limited to lock and unlock only pathnames
that contain no spaces. eg: folders such as c:\program files cannot be locked.
But the latter versions of this software will overcome these difficulties.
even the locked folders cannot be accessed by any system processes.
